# Cut To Size Interiors Website

Marketing site for Cut To Size Interiors built with Next.js, TypeScript, Tailwind, and Motion for smooth scroll-triggered animations.

## Getting Started

1.  **Install dependencies:**

    ```bash
    pnpm install
    ```

2.  **Run the development server:**

    ```bash
    pnpm dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies

- Next.js
- TypeScript
- Tailwind CSS
- Motion.dev
